# BIOC4605
This thesis investigated whether spider silk protein sequences could predict biome class and identify sequence features associated with environmental variation. Using sequence and metadata from the Spider Silkome Database, biome-labelled datasets were analyzed in two stages. Initially, models were created using the Worldwide Fund for Nature (WWF) and Whittaker biome classifications to establish baseline behaviour for the model and optimize training parameters. These pooled analyses showed that strong class imbalance limited ecological interpretation, particularly in the WWF-labelled dataset, where accuracy was driven by dominant biome classes. Subsequently, the Whittaker-labelled data were separated into seven silk-specific subsets: AcSp, AgSp, CySp, Flag, MaSp, MiSp, and PySp. Flag showed the highest accuracy (50.00%), whereas MaSp and CySp showed the weakest performance (29.69% and 29.44%, respectively). The top motifs also differed by silk type and emphasized local residue chemistry instead of the motifs associated with each silk type. Thus, these results show that spider silk sequences contain detectable biome-associated signals, but the strength and interpretability of these signals depend strongly on silk type, dataset balance, and sequence representation. This work provides a proof-of-concept for linking spider silk sequence variation to ecological context and establishes a foundation for future mechanistic and recombinant silk design studies.
